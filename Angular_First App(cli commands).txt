Angular Commands :
   1. To create the new angular file --- (ng new Fliename --style=scss --routing --skip-install)/ ng new Filename
   2. For packages -- npm install
   3. To open angular application in the browser locally --  ng serve --open --port=5200(change the port number)
		You can use either of --port 5200 and --port=5200 for specifying the port number

   Note : All angular commands start from 'ng'..

Adding Bootstrap: 

    1. npm install jquery --save
    2. npm install popper.js --save
    3. npm install bootstrap --save
    4. npm install font-awesome --save


Note : if bootstrap is not working , then follow this
	insert this line -- @import "~bootstrap/dist/css/bootstrap.css" in style.css
